# c语言

## 初识C语言

### 1.第一个C语言程序

~~~ c
// c语言代码中一定要有main函数
// printf是一个库函数，专门用来打印数据
// std = 标准
// i = input 输入
// o = output 输出
// main是程序的入口
// main函数有且仅有一个
#include <stdio.h>

int main() {
	printf("hello\n");
	return 0;
}
~~~
